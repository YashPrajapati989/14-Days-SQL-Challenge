## 📖 About
A 14-day SQL learning journey featuring practical business problems, datasets, analytical SQL queries, window functions, aggregations, joins and reporting. This repository documents daily progress and showcases SQL skills for data analytics portfolios.

## ✨ Highlights
- Beginner to Advanced SQL
- Real-world datasets
- Business analytics questions
- Clean SQL scripts
- CSV outputs and reports
- Portfolio-ready project

## 📚 Topics
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- JOINS
- CASE
- Aggregate Functions
- CTEs
- Window Functions
- ROW_NUMBER, RANK, DENSE_RANK
- LEAD, LAG
- FIRST_VALUE, LAST_VALUE
- NTILE

## 🚀 Getting Started
1. Clone the repository.
2. Import the CSV datasets into MySQL.
3. Execute each day's SQL file in sequence.
4. Compare your results with the provided outputs.
